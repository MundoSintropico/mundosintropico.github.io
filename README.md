# CREACIÓN DE TRANSPARENCIA

## Objetivo
Este repositorio recoge el proyecto **CREACIÓN DE TRANSPARENCIA**, una iniciativa destinada a desvelar los mecanismos invisibles que sostienen nuestros actos cotidianos. El objetivo es mostrar el contraste entre lo visible (un gesto trivial, como pagar con tarjeta) y las capas ocultas que lo hacen posible: contratos, algoritmos, cadenas de datos, implicaciones sociales y ecológicas. Buscamos transformar la opacidad en claridad, invitando a reflexionar sobre el significado profundo de aquello que damos por hecho【275689903118586†L160-L182】.

## Cómo participar
Para colaborar en este proyecto puedes proponer y documentar nuevos casos de estudio. Cada caso debe incluir:

* **Acto cotidiano**: una acción sencilla y reconocible (por ejemplo, subir una foto, decir «ok» a una app o encender una luz inteligente).
* **Plano invisible**: aspectos ocultos activados por esa acción (metadatos, contratos, vigilancia algorítmica, externalidades ambientales, redes de procesamiento, etc.).
* **Tono narrativo**: decide si la representación será filosófica, educativa, poética, visual o distópica.

Los análisis pueden representarse mediante narrativas, guiones para vídeo corto, gráficos o imágenes generativas. Por favor, guarda tus contribuciones en carpetas apropiadas (por ejemplo, `casos/acto-inicial`, `guiones/`, `imagenes/`) y utiliza nombres descriptivos de archivo.

## Variables del proyecto
En este proyecto trabajamos con tres variables principales. A continuación se muestran ejemplos (de forma resumida) para inspirarte:

| Variable           | Ejemplos                          |
|--------------------|------------------------------------|
| Acto cotidiano     | Pagar con tarjeta, subir una foto, usar reconocimiento facial |
| Plano invisible    | Bancos y cláusulas, metadatos y geolocalización, algoritmos de vigilancia |
| Tono narrativo     | Educativo, filosófico, poético, distópico |

## Análisis de entropía/sintropía
Utilizamos un enfoque inspirado en la teoría de sistemas complejos para evaluar cómo determinadas acciones humanas generan **entropía** (desorden, incertidumbre) o **sintropía** (orden creativo, cooperación). Para ello:

1. **Resumen del contexto**: describe brevemente el sistema en el que ocurre la acción (organización, comunidad, mercado, red social).  
2. **Identificación de acciones**: lista las conductas clave y clasifícalas como entrópicas o sintrópicas.  
3. **Métricas**: asigna a cada acción puntuaciones de 0 a 10 en aspectos como orden estructural, cooperación, cohesión, sostenibilidad y calidad de información.  
4. **Índices**: interpreta el equilibrio global entre entropía y sintropía, señalando focos de desorden y vectores de creación de valor.  
5. **Intervenciones**: propone cambios concretos (reglas, incentivos, flujos de información) para reducir entropía y potenciar sintropía.

Puedes consultar la escala de análisis completa en [nakamotobook.com](https://nakamotobook.com/la-civilizacion-en-2026-2036-colapso-o-bifurcacion/). Recuerda ser explícito cuando falte información y señalar el nivel de confianza de tus conclusiones.

## Estructura del repositorio
- `casos/` – Ejemplos y descripciones de actos cotidianos analizados, cada uno en su propia carpeta.  
- `analisis/` – Documentos de análisis de entropía/sintropía y sugerencias de intervenciones.  
- `guiones/` – Guiones narrativos o propuestas de vídeos cortos basados en los casos.  
- `imagenes/` – Recursos visuales o imágenes generadas para acompañar las narraciones.  
- `doc/` – Documentación adicional, referencias y materiales de apoyo.

## Contribución
Este proyecto está abierto a contribuciones. Para participar:

1. **Crea un fork** de este repositorio y clona tu copia local.  
2. Añade tus casos o análisis en las carpetas correspondientes siguiendo la estructura indicada.  
3. Describe claramente el acto, el plano invisible y el tono; incluye tus hallazgos sobre entropía/sintropía y cualquier recurso visual o narrativo.  
4. Haz un *pull request* con tus cambios, explicando brevemente qué aportan.
